# 数组

数组是多个相同类型数据的组合，实现对这些数据的统一管理。可以看成是一个容器。

数组中的元素可以是任何数据类型，包括基本数据类型和引用数据类型。数组元素类型决定了数组的数据类型。

数组属于引用类型，数组是对象，数组中的每个元素相当于该对象的成员变量。

数组一旦初始化，其长度是不可变的。

## 一维数组的创建及使用

一维数组实质上是一组相同数据类型数据的线性集合，当程序中需要批量操作某种相同数据类型的数据时可以使用它。

### 定义（声明）

数组元素类型[] 数组名字;

数组元素类型 数组名字[];

public class TestArray {    public static void main(String[] args) {        int[] i;        int a[];        char[] c;        boolean[] b;        String[] s;        Mydate[] c; // 对象数组    }}

此时只是声明了数组名字和要存放元素的数据类型，还没有分配内存空间存放任何数据。

### 初始化

对数组进行初始化，即为数组中的元素分配内存空间和赋值。对数组名进行初始化，即给数组名赋值为数组对象的引用。

数组在初始化时必须指定长度

堆内存的数组空间长度一旦确定，不能再次发生改变

栈内存的变量中存储的是数组的地址引用

一共存在两种初始化方式：静态初始化和动态初始化。

#### 静态初始化

静态初始化是指由程序员在初始化数组时为数组每个元素赋值，由系统决定数组的长度。

静态初始化有两种方式可以实现：

public class TestArray {    public static void main(String[] args) {        int[] array1 = new int[]{10, 20, 30, 40, 50};        int[] array2 = {10, 20, 30, 40, 50}; // 简便，推荐！        MyDate dates[] = {            new MyDate(22, 7, 1964),            new MyDate(1, 1, 2000),            new MyDate(22, 12, 1964)		}    }}

注意：只有在定义数组的同时执行数组初始化才支持使用简化的静态初始化。

#### 动态初始化

动态初始化是指由程序员在初始化数组时指定数组的长度，由系统为数组元素分配初始值。

格式：

数据类型[] 数组名 = new 数据类型[数组长度];

示例:

public class TestArray1 {    public static void main(String[] args) {        int[] array = new int[10];        array[0] = 3;        array[1] = 0;        for (int value : array) {            System.out.println(value);        }                // [ 代表是数组，有几个就是几维数组        // I 代表是 int 类型        // @ 是固定的        // 2d98a335 代表的是数组的地址值        System.out.println(array); // [I@2d98a335                        MyDate dates[];        dates = new MyDate[3];        dates[0] = new MyDate(22, 7, 1964);        dates[1] = new MyDate(1, 1, 2000);        dates[2] = new MyDate(22, 12, 1964);    }}

在数组声明的同时，系统会分配一块内存空间供该数组使用，其中数组长度是10，由于每个元素都为 int 型数据类型，因此上例中数组占用的内存共有10*4=40个字节。此外，动态初始化数组时，会根据它的数据类型设置为不同类型的默认初始值。本例数组中每个元素的默认值为0，其他常见的数据类型默认值如下表所示：

整数类型：byte,short,int,long默认初始化值都是0

浮点类型：float,double默认初始化值都是0.0

布尔类型：boolean默认初始化值false

字符类型：char默认初始化值'\u0000'，也就是一个空格。（\u0000,每一个0其实代表的是16进制的0,那么四个0就是代表16个二进制位）

引用数据类型：默认值初始值为null

## 二维数据的创建

先声明，再用 new 运算符进行内存分配：

//方式一int arr[][];//int[][] arr;arr = new int[2][4];//方式二int arr1[][];arr1 = new int[2][];arr1[0] = new int[4];arr1[1] = new int[4];

初始化：

int arr[][] = {{1, 2}, {3, 4}};

## 数组元素的访问

数组的数据可以用索引来访问，索引从0开始，到数组的 length 属性减1结束。当提供的索引超出了范围，会出现一个运行时异常。

public class TestArray1 {    public static void main(String[] args) {        int[] arr = new int[]{10, 20, 30, 40, 50};        int value = arr[6];        System.out.println(value); // Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: Index 6 out of bounds for length 5 at TestArray1.main(TestArray1.java:5)    }}

有时我们需要取出所有元素然后进行一些处理，显然上面通过索引的方式略显麻烦，此时使用遍历会事半功倍。

因为数组属于引用类型，所以当数组作为参数传递或者作为方法返回值时，操作的都是内存地址。
