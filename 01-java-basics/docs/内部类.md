# 内部类

定义在类中的类，分为成员内部类和局部内部类。

成员内部类是定义在类中的，局部内部类是定义在方法中的。

成员内部类又可以定义为静态和非静态的。两种类的调用方式：

public class InnerClassTest {    public static void main(String[] args) {        Person person = new Person();        Person.Dog dog = person.new Dog();        dog.eat();        Person.Bird bird = new Person.Bird();        bird.eat();    }}class Person{    //非静态的    class Dog{        public void eat(){            System.out.println("Dog eat...");        }    }    //静态的    static class Bird{        public void eat(){            System.out.println("Bird eat...");        }    }}
