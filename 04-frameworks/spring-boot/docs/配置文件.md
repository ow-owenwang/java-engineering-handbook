# 配置文件

## 配置管理

### 使用环境变量进行配置

配置文件中的配置项可以通过环境变量进行配置，如：

server:  port: ${MY_PORT}

在IDEA中配置端口：

但这样做也可能会带来一些问题，如打包时，maven clean install会报错，跳过test即可。再有，就是启动jar包时，需要指定端口java -jar spring-boot-demo-0.0.1-SNAPSHOT.jar --MY_PORT=80。

![](assets/document_image_rId4.png)

### 引用外部文件进行配置

java -jar spring-boot-demo-0.0.1-SNAPSHOT.jar命令默认会使用与jar包相同路径的application.yml配置文件，利用这一点可以快速对应用进行配置。

### 使用命令行参数进行配置

换成命令行中启动就是java -jar spring-boot-demo-0.0.1-SNAPSHOT.jar --server.port=9000。
