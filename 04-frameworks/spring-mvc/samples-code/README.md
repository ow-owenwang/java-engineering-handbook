# samples-code

本目录旨在存放各个特性的实际运行代码。

## 示例列表

- `mvc-basic`: 包含基本的控制器、请求映射、视图解析。
- `mvc-rest`: 包含 RESTful API、JSON 响应、异常处理。
- `mvc-interceptor`: 包含拦截器的实现与配置。

## 如何运行

1. 确保安装了 JDK 17+ 和 Maven 3.6+。
2. 进入具体子项目目录。
3. 执行 `mvn spring-boot:run` (本知识库案例采用 Spring Boot 简化部署)。

> 注：虽然 Spring MVC 可以脱离 Spring Boot 运行，但为了快速复现和学习，推荐使用 Spring Boot 进行实验。
